# 谈一下我对某些反对「米游一家亲」的声音的看法

> bilibili [@纯美忆者](https://space.bilibili.com/3546660716153442)

首先，我们先明确一个概念，在互联网上，每个人的观念都是不同的，互联网社区是由一个个不同的交集构成，你喜欢吃肉馅烧卖，我喜欢吃糯米馅的，但是不妨碍我们都玩米游，都喜欢构建并维护游戏社区。

我们先讲讲什么是一家亲。

当我们在各种米游评论区里看到赛诺闭嘴，三月七合影留念，小黄鸭销冠玩梗时，是玩家一家亲吗？

当原铁绝项目组上台领奖时，台下其它组的同事举牌助威，是米游一家亲吗？

当崩铁的佩拉手机上印着未定男主，未定事件簿里点缀着不少原崩的元素，是米游一家亲吗？

当有米游被黑潮入侵，各个游戏的玩家都赶来支援，发美图抵御攻击，是米游一家亲吗？

一家亲是绝大部分米游玩家认可的共识，某些节奏社区的玩家，不管是什么身份，都没有资格代替大部分人切割一家亲。当然，其他人也没有资格按着他们去支持一家亲。

毕竟，在互联网上，没人有资格要求别人和自己观点完全一致。道同则合，道不同则陌路。

或许大家观点有分歧，但至少我们都是原神玩家，又或者崩铁玩家，未定玩家……我们还是有着很多共同语言。

当然，若是某些人反「一家亲」入脑，借着这个旗号对各种自己不喜欢米家游戏展开无底线攻击，那他和那些米黑又有什么区别呢？用着米黑的话术，干着米黑做的事，这种到时候该批判就批判，该辩经就辩经。

以上，就是我对这类声音的全部看法。
